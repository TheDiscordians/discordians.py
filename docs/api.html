The discordians API reference¶

Client¶

class discordians.client.DiscordiansClient(loop=None)¶

The main class to interact with the API.

Parameters:loop (Optional[EventLoop]) – The event loop to use for HTTP requests. Got using asyncio.get_event_loop()
loop¶

EventLoop – The event loop, the client uses to make HTTP requests.

sessionClosed¶

bool – Indicates if the internal HTTP session is closed.

coroutine close()¶

Closes the internal HTTP session.

This function should be always called before closing the program.

coroutine cursive(*, text=None)¶

Return cursive text from normal text.

Parameters:

text (str) – The text to convert.
Returns:

A dictionary that contains the cursive text.

{
    "message": The converted text.
}

Return type:

dict
Raises:
  • ArgumentError – Invalid arguments passed.
  • DifferentResponseCode – The API returned a non-200 response code.

Exceptions¶

These are the exceptions raised by the client.

exception discordians.exceptions.DifferentResponseCode¶

Raised when the API returns a different status code. (non-200).

exception discordians.exceptions.ArgumentError¶

Raised when the supplied argument to a method does isn’t right.
